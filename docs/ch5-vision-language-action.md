## Spec-Kit Plus Block

- **Learning Objectives**
  - Describe VLA architectures for robotics.
  - Connect camera, speech, and action spaces.
  - Implement a minimal perception–language–action pipeline over ROS 2.

- **Required Skills**
  - Python, basic deep learning, REST APIs.

- **System Components**
  - Vision encoders, language models, action decoders.

- **Inputs & Outputs**
  - Inputs: images, text or speech.
  - Outputs: action goals, trajectories.

- **Tools & Frameworks**
  - PyTorch or similar, speech-to-text, text-to-speech.

- **Performance Constraints**
  - Sub-second VLA response times for interaction.

## Chapter Body

_This chapter will be expanded with end-to-end VLA pipelines, including ROS 2 topic interfaces, Python code, and safety wrappers for deploying language-driven humanoid behaviors._
