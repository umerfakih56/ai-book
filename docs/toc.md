# Physical AI & Humanoid Robotics - Table of Contents

## Overview

This comprehensive guide covers building intelligent humanoid robots using modern robotics and AI technologies.

## Modules

### The Robotic Nervous System (ROS 2)

Master the Robot Operating System 2 framework for building distributed robotic applications

**Duration**: 40 hours | **Difficulty**: intermediate

1. [ROS 2 Architecture & Concepts](module-1/chapter-1.mdx)
   - Introduction to ROS 2 framework, its architecture, and core concepts for robotic development
   - Duration: 10h | Difficulty: beginner

2. [Nodes, Topics, Services & Actions](module-1/chapter-2.mdx)
   - Deep dive into ROS 2 communication patterns including publish/subscribe, request/response, and goal-based interactions
   - Duration: 12h | Difficulty: intermediate

3. [URDF & Robot Description Formats](module-1/chapter-3.mdx)
   - Learn to create robot models using Unified Robot Description Format and visual representations
   - Duration: 10h | Difficulty: intermediate

4. [Python rclpy Programming](module-1/chapter-4.mdx)
   - Develop complete ROS 2 applications using Python client library rclpy
   - Duration: 8h | Difficulty: intermediate
