# Course Assessments

This course uses a comprehensive assessment system to evaluate your progress and skills across all learning modules.

## Assessment Schedule

| Week | Assessment | Type | Points | Weight |
|------|------------|------|--------|--------|
| 5 | [ROS 2 Basics Quiz](ros2-basics-quiz.mdx) | Quiz | 100 | 15% |
| 7 | [Gazebo Project - Build Custom Robot](gazebo-robot-project.mdx) | Project | 150 | 20% |
| 10 | [Isaac SLAM Implementation Project](isaac-slam-project.mdx) | Project | 200 | 25% |
| 13 | [VLA Capstone - Autonomous Humanoid](vla-capstone.mdx) | Capstone | 300 | 40% |

## Assessment Types

### Quizzes

Quizzes evaluate your understanding of theoretical concepts and fundamental knowledge:

- **[ROS 2 Basics Quiz](ros2-basics-quiz.mdx)** (Week 5, 100 points)

### Projects

Projects assess your practical skills and ability to apply concepts to real-world problems:

- **[Gazebo Project - Build Custom Robot](gazebo-robot-project.mdx)** (Week 7, 150 points)
- **[Isaac SLAM Implementation Project](isaac-slam-project.mdx)** (Week 10, 200 points)

### Capstone

The capstone project evaluates your ability to integrate all course technologies:

- **[VLA Capstone - Autonomous Humanoid](vla-capstone.mdx)** (Week 13, 300 points)

## Grading Scale

- **A**: 90-100%
- **B**: 80-89%
- **C**: 70-79%
- **D**: 60-69%
- **F**: 0-59%

## Submission Guidelines

**Late Policy**: 10% penalty per day late, maximum 5 days

**Academic Integrity**: All work must be original. Proper citation required for external resources.

**Collaboration**: Discussion encouraged, but submission must be individual work.

**Format Requirements**: Follow specified submission formats exactly. Include all required deliverables.

## Certificate Requirements

**Minimum Grade**: 70%

**Required Assessments**:
- ROS 2 Basics Quiz
- Gazebo Project - Build Custom Robot
- Isaac SLAM Implementation Project
- VLA Capstone - Autonomous Humanoid
