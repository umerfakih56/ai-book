# Introduction to Physical AI & Humanoid Robotics

Welcome to this comprehensive guide to building intelligent humanoid robots using cutting-edge technologies including ROS 2, simulation platforms, and AI systems.

## About This Course

This course provides a complete learning path from basic robotics concepts to advanced AI-powered humanoid robots. You will learn:

- **ROS 2 Fundamentals**: Master the Robot Operating System 2 framework
- **Simulation Technologies**: Work with Gazebo, Unity, and NVIDIA Isaac Sim
- **AI Integration**: Implement vision, language, and action systems
- **Practical Projects**: Build real-world robotic applications

## Course Structure

The course is organized into four comprehensive modules:

1. **The Robotic Nervous System (ROS 2)** - Learn the foundation of modern robotics
2. **The Digital Twin (Gazebo & Unity)** - Master simulation environments
3. **The AI-Robot Brain (NVIDIA Isaac)** - Implement advanced AI systems
4. **Vision-Language-Action (VLA)** - Create intelligent, interactive robots

## Learning Paths

Choose the path that best fits your goals:

- **Complete Physical AI**: Comprehensive 13-week program
- **Simulation Specialist**: Focus on simulation technologies
- **Software Developer**: Emphasis on programming and AI
- **Quick Start**: Essential concepts for rapid deployment
- **Robotics Foundations**: Strong fundamentals for beginners

## Getting Started

Begin with Module 1 to build your ROS 2 foundation, or choose a learning path that matches your experience level and goals.

Let us start your journey into the fascinating world of physical AI and humanoid robotics!
