Simulate various robot sensors including LiDAR, cameras, and inertial measurement units

## Learning Outcomes

- Configure LiDAR sensors in simulation
- Set up camera systems with realistic parameters
- Implement IMU and GPS sensors
- Generate sensor data for algorithm testing

- Signal processing fundamentals
